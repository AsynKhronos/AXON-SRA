# Branding assets

## `axon-mark.svg`

Primary GitHub project mark.

Properties:

- true vector SVG;
- circular white field;
- dark geometric A/X monogram;
- no square raster background;
- suitable for GitHub light and dark themes.

## `axon-sra-social-preview.png`

Prepared social-preview artwork.

Upload through:

```text
Repository → Settings → General → Social preview → Edit → Upload an image
```

The preview is not a GitHub Pages asset and does not create a separate website.
