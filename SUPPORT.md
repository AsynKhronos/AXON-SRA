# Support

AXON-SRA is currently a research prototype.

## Use GitHub Issues for

- reproducible defects;
- compatibility reports;
- semantic-analysis regressions;
- documentation corrections;
- bounded research proposals.

## Do not use GitHub Issues for

- general DXVK support unrelated to AXON-SRA;
- unsupported binary downloads;
- proprietary SDK redistribution requests;
- speculative “support every game” requests without evidence;
- private security disclosures.

Read the [documentation index](docs/README.md) and [compatibility policy](docs/compatibility.md) before opening an issue.
